# 家庭囤货助手 v1.0 - APK构建说明

## 📱 最终版本构建

### 构建方式选择

对于**分享给同学使用**的APK，推荐使用 **Preview模式**：

#### ✅ Preview模式（推荐）
- **格式**：APK（可直接安装）
- **适用场景**：分享给朋友、同学使用
- **优势**：
  - 包含所有完整功能
  - 无需Google Play签名
  - 可以直接在手机上安装
  - 适合非商业分享

#### ❌ Production模式（不推荐）
- **格式**：AAB（需Google Play）
- **适用场景**：上架应用商店
- **限制**：需要复杂配置，不适合直接分享

## 🚀 一键构建

### 步骤1：运行构建脚本
```bash
# 双击运行 build_final.bat 文件
# 或在命令行中运行：
build_final.bat
```

### 步骤2：等待构建完成
- 构建时间：约5-15分钟
- 构建过程中会显示进度
- 请保持网络连接

### 步骤3：下载APK
构建完成后，访问：
```
https://expo.dev/accounts/suxian/projects/stock-manager
```
- 点击最新的构建记录
- 下载APK文件
- 传输到Android手机安装

## 📋 构建配置说明

### eas.json 配置
```json
{
  "build": {
    "preview": {
      "android": {
        "buildType": "apk"  // 确保是APK格式
      }
    }
  }
}
```

## 📱 安装说明

### Android手机安装
1. 允许"未知来源"安装
2. 点击下载的APK文件
3. 按照提示完成安装

### 首次运行
1. 允许相机和存储权限
2. 开启通知权限（用于过期提醒）

## 🛠️ 故障排除

### 构建失败
- 检查网络连接
- 确认Expo账户权限
- 查看控制台错误信息

### 安装失败
- 检查APK文件完整性
- 确认Android版本兼容性
- 尝试重启手机后安装

## 📞 技术支持

如有问题，请检查：
1. Expo账户是否正确登录
2. 网络连接是否正常
3. 构建日志中的错误信息
