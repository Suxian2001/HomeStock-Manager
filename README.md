# 家庭囤货助手 v1.0

一个简洁高效的物品管理应用，帮助您轻松管理家庭用品的库存和过期情况。

## ✨ 功能特性

- 📦 **物品管理**：添加、编辑、删除物品
- 📅 **过期提醒**：智能过期状态显示
- 📸 **图片上传**：支持拍照和相册选择
- 🔍 **智能筛选**：按分类、位置、状态筛选
- 📊 **统计信息**：实时显示过期物品统计
- 🎨 **现代化UI**：简洁直观的用户界面

## 🚀 快速开始

### 开发环境
```bash
npm install
npm start
```

### 构建APK
```bash
# 使用提供的构建脚本
build_final.bat

# 或手动构建
set EAS_NO_VCS=1
npx eas build --platform android --profile preview
```
