# 📱 家庭囤货助手 - 图标配置指南

## 🎯 只需要3个文件！

只需准备3个PNG图片文件，放在正确位置即可！

## 📁 文件位置和命名

**位置**: `项目根目录/assets/` 文件夹

### 1. 主图标 → `icon.png`
- **尺寸**: 1024×1024像素
- **用途**: App图标、桌面图标
- **要求**: 正方形，有内容区域

### 2. Android图标 → `adaptive-icon.png`
- **尺寸**: 1024×1024像素
- **用途**: Android自适应图标
- **要求**: 单色设计，透明背景

### 3. 启动画面 → `splash.png`
- **尺寸**: 1242×2436像素
- **用途**: App启动时的加载画面
- **要求**: 包含logo和文字

## 🎨 设计建议

### 图标主题
- **颜色**: 蓝色 (#3b82f6)
- **元素**: 箱子、物品、存储相关
- **风格**: 简洁现代

### 示例设计
```
   ┌─────────┐
   │  📦📦📦  │
   │ 囤货助手 │
   │         │
   └─────────┘
```

## ⚡ 快速创建方法

### 方法1: 使用AI工具 (推荐)
1. 打开: https://www.bing.com/images/create 或 Midjourney
2. 输入: "blue storage box icon with chinese text 囤货助手"
3. 生成图片后调整尺寸

### 方法2: 使用在线图标生成器
1. 访问: https://icon.kitchen/
2. 搜索: "storage" 或 "box"
3. 选择蓝色主题
4. 导出1024x1024 PNG

### 方法3: 手动创建
1. 用画图工具或Photoshop
2. 创建1024x1024画布
3. 绘制箱子图标
4. 添加"囤货助手"文字
5. 保存为PNG

## ✅ 完成步骤

1. **准备3个PNG文件**（按上面命名）
2. **放入 `assets/` 文件夹**
3. **重新构建APK**
4. **享受自定义图标！**
